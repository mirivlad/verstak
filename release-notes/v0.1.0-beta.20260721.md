## Highlights

- Improved beta-readiness across the desktop shell: settings panels now use
  the available modal width with comfortable padding, filter groups wrap as a
  unit, and empty Deal overview surfaces expand into the available workspace.
- Global search now publishes current progressive results instead of showing a
  premature empty state while providers are still running.
- Sync status now distinguishes configuration, active work, success, and
  failure, and reports the last successful sync together with sent, received,
  and queued operation counts.
- Large sync queues are uploaded in server-compatible batches. A queue of 103
  operations now completes without the previous HTTP 413 failure.
- Plugin workspace discovery now exposes only semantic Deals where the calling
  plugin is active, preventing Browser Inbox assignments to invisible targets.

## Главное

- Улучшена готовность desktop-приложения к бета-версии: панели настроек
  используют доступную ширину модальных окон с нормальными отступами, группы
  фильтров переносятся целиком, а пустой обзор Дела занимает свободную область.
- Глобальный поиск теперь показывает актуальные промежуточные результаты и не
  выводит преждевременно сообщение об отсутствии результатов.
- Статус синхронизации различает настройку, выполнение, успех и ошибку, а также
  показывает время последней успешной синхронизации и количество отправленных,
  полученных и ожидающих операций.
- Большие очереди синхронизации отправляются пакетами, совместимыми с лимитом
  сервера. Очередь из 103 операций больше не завершается ошибкой HTTP 413.
- Плагины получают только семантические Дела, в которых активен вызывающий
  плагин, поэтому Browser Inbox больше не предлагает невидимые назначения.

## Packages / Пакеты

This prerelease provides a portable Windows archive, a Debian package, and an
AppImage for Linux. Official plugins are bundled with the desktop packages.

В этот prerelease входят переносимый архив для Windows, Debian-пакет и AppImage
для Linux. Официальные плагины включены в desktop-пакеты.
