## Highlights

- File and note renaming works again for unique names while real destination conflicts remain protected.
- Global search correctly indexes directories containing exactly two entries and no longer shows archived Browser Inbox captures after they have been converted.
- Converted notes and files are searchable from their current vault location instead of being represented by stale browser material.
- The global search results panel is approximately three times wider than the input and reserves enough height for at least five results while remaining responsive on small windows.
- File duplication now chooses the first available copy name without relying on the broken metadata preflight.

## Главное

- Переименование файлов и заметок снова работает для свободных имён, а реальные конфликты назначения по-прежнему безопасно блокируются.
- Глобальный поиск корректно индексирует каталоги ровно с двумя элементами и больше не показывает архивированные материалы Browser Inbox после преобразования.
- Созданные из браузерных материалов заметки и файлы находятся по их актуальному расположению в хранилище, без устаревшего результата из инструмента «Браузер».
- Панель результатов глобального поиска стала примерно втрое шире поля ввода и вмещает не менее пяти результатов, сохраняя адаптивность в небольших окнах.
- Команда создания копии файла теперь выбирает первое свободное имя без ошибочной предварительной проверки метаданных.

## Packages / Пакеты

This patch prerelease provides an updated portable Windows archive, Debian package, and AppImage for Linux. Updated official plugins are bundled with the desktop packages.

Этот patch-prerelease включает обновлённый переносимый архив для Windows, Debian-пакет и AppImage для Linux. Обновлённые официальные плагины включены в desktop-пакеты.
