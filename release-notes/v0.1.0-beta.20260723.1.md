## Highlights

- Todo now discovers semantic Deals instead of ordinary vault folders and only shows Deals where the Todo tool is enabled.
- Deal creation exposes an exact, independently selectable tool set for every template, including an empty Custom starting point.
- Default Editor wraps long lines by default and persists the wrap toggle without changing saved text or newline content.
- Files can safely copy or move files and folders between Deals. Drops on a Deal use its existing `Files` directory and report missing support or conflicts without creating partial destinations.
- Sidebar context menus render above clipping boundaries, the sidebar width is bounded and persistent, and double-clicking the divider restores the default.
- Tree drag-and-drop uses stable UUID identities with before, inside, after, and root targets, hover expansion, edge autoscroll, and complete cancellation cleanup.
- User-defined mixed folder/Deal sibling order is stored as internal vault metadata and synchronized through a dedicated order entity. Device-local width, scroll, expansion, and selection state remain excluded.

## Главное

- Todo теперь получает семантические Дела, а не обычные папки vault, и показывает только Дела с включённым инструментом Todo.
- Создание Дела позволяет независимо выбрать точный набор инструментов для любого шаблона, включая пустую отправную точку Custom.
- Default Editor по умолчанию переносит длинные строки и сохраняет переключатель переноса, не меняя текст и исходные переводы строк.
- «Файлы» безопасно копируют и перемещают файлы и папки между Делами. Перетаскивание на Дело использует существующую папку `Files`, а отсутствие поддержки и конфликты показываются без частично созданных данных.
- Контекстные меню sidebar больше не обрезаются, ширина sidebar ограничена и сохраняется, а двойной щелчок по разделителю возвращает значение по умолчанию.
- Перетаскивание в дереве использует стабильные UUID, цели до/внутрь/после/в корень, раскрытие по наведению, автопрокрутку у края и полную очистку при отмене.
- Пользовательский смешанный порядок папок и Дел хранится во внутренних метаданных vault и синхронизируется отдельной сущностью. Локальные ширина, прокрутка, раскрытие и выбор между устройствами не передаются.

## Packages / Пакеты

This patch prerelease provides a portable Windows archive, a Debian package, and an AppImage for Linux. Matching official plugins are bundled with every desktop package.

Этот patch-prerelease включает переносимый архив для Windows, Debian-пакет и AppImage для Linux. Соответствующий комплект официальных плагинов включён в каждый desktop-пакет.
